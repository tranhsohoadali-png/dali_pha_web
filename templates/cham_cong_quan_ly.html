Hướng dẫn cho bên kế toán cần làm gì (bấm xem)
Bên ketoan.tranhdali.vn tạo 1 endpoint GET. App này sẽ gọi tới:
GET {URL}?key=...&user=<tên đăng nhập>&day=YYYY-MM-DD&month=YYYY-MM
Endpoint trả về JSON đúng dạng: 
{
  "ok": true,
  "day": 250000,          // lương ngày hôm đó (đồng)
  "month": 5000000,       // tổng lương tháng tới hiện tại
  "month_label": "06/2026"
}
  • Kiểm tra key khớp khoá đã lưu ở trên; sai thì trả {"ok":false,"error":"sai key"}.
  • user = tên đăng nhập nhân viên (giống bên chấm công).
  • Tên trường linh hoạt: chấp nhận day/luong_ngay, month/luong_thang/total.
